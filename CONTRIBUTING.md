# Contributing to NotumAi

Thank you for your interest in contributing to NotumAi! Open source contributions are what make the community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone the project** to your local machine:
   ```bash
   git clone https://github.com/your-username/notum-ai.git
   cd notum-ai
   ```
3. **Create a new branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/AmazingFeature
   ```

## Project Structure

- `ElectronFrontend/`: Contains the Electron-based user interface.
- `PythonServer/`: Contains the Python backend (FastAPI) and logic for AI models (SAM 2).

## Development Setup

Please refer to the `README.md` for detailed installation and setup instructions for both the frontend and backend components.

## Submitting Changes

1. **Make your changes** in your feature branch.
2. **Commit your changes**:
   ```bash
   git commit -m 'Add some AmazingFeature'
   ```
3. **Push to the branch**:
   ```bash
   git push origin feature/AmazingFeature
   ```
4. **Open a Pull Request** against the `main` branch.

## Guidelines

- **Code Style**: Please maintain consistency with the existing code style.
- **Documentation**: Update documentation if you are adding new features or changing existing functionality.
- **Testing**: Ensure your changes do not break existing functionality. We encourage adding tests for new features.
- **Issues**: If you find a bug, please open an issue first to discuss it.

## Code of Conduct

Please note that this project is released with a [Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.

Happy coding!
